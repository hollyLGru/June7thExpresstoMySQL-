API Documentation

Get /Todos

this route can be used to retrieve a todo item . but only the id, is_done, task are included

GET /Todos/:id

We can retrieve a todo based on the ID for that specific to do

Post /Todos

This route can be used to create a todo item. The body of the request should include the task description

Delete /Todos

This route will delete the Todo based on ID

Update /Todos

We can change a todo item with this route. We need an ID with this to know what todo to change
